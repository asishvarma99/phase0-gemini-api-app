# Phase 0 Gemini API App

A terminal-based cloud AI chatbot built with Python and the Gemini API.

## Features

- Connects Python to the Gemini cloud API
- Accepts interactive user prompts
- Maintains short-term conversation history
- Supports clearing the current conversation
- Handles missing keys and API errors
- Uses environment variables for secure authentication

## Tech Stack

- Python
- Gemini Developer API
- Google GenAI SDK
- python-dotenv
- Git and GitHub
